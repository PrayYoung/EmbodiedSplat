## Embodied Splat Sim

**Embodied Splat Sim** is a photorealistic reinforcement learning environment for embodied agents built on 3D Gaussian Splatting.

The project explores how agents can learn navigation and spatial reasoning directly from image observations rendered from real-world scenes.
